Génère un template de message direct OnlyFans selon le contexte fourni par Leo.

Demande d'abord :
1. Type de DM : relance abonné inactif / upsell PPV / réponse standard / autre
2. Profil de la créatrice (style, niche)
3. Cible (nouvel abo, abo long, big spender, etc.)

Puis propose 2-3 variantes courtes (50-150 caractères max), avec :
- Ton naturel, pas vendeur
- Accroche personnalisée
- CTA clair mais subtil

Format de sortie : variantes numérotées, sans explications longues.
