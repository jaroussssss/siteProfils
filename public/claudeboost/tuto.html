Installation en 5 minutes

Suis les étapes — tout est automatique.

📂 Ce qui va être installé sur ton ordinateur
C:\Users\[toi]\ClaudeSetup\Scripts et profil Claude
C:\Users\[toi]\.claude\Config et CLAUDE.md
C:\Users\[toi]\AppData\Roaming\npm\Claude Code CLI
~/ClaudeBoost/Scripts et profil (modifiable au démarrage)
~/.claude/Config et CLAUDE.md
~/.npm-global/bin/Claude Code CLI
/opt/homebrew/Homebrew + Node.js (si installés)
Le script proposera ~/ClaudeBoost comme répertoire par défaut — tu peux entrer un autre chemin si tu préfères.
Pour désinstaller : supprime ces dossiers et relance le script de ton gestionnaire de paquets.
1
Ouvre PowerShell en administrateur
Touche Windows → tape PowerShell → clic droit → "Exécuter en tant qu'administrateur"
💡 Tu dois voir "Administrateur" dans la barre de titre de la fenêtre.
2
Installe Node.js si ce n'est pas fait
Node.js est nécessaire pour faire tourner Claude Code. Si tu ne sais pas si tu l'as, va sur nodejs.org et télécharge la version LTS.
⚠️ Redémarre le PowerShell après l'installation de Node.js.
3
Clique "Copier" et colle dans PowerShell
La commande installe tout automatiquement — pas de téléchargement de fichier, pas de blocage Windows.
Set-ExecutionPolicy Bypass -Scope Process -Force; irm https://tempcestlouisquirac.fr/claudeboost/downloads/install-leo.ps1 | iex
⏳ L'installation dure 1 à 3 minutes. Tu verras des messages s'afficher.
4
Ferme et rouvre le terminal
Ferme PowerShell complètement, puis rouvre un terminal normal (pas forcément admin cette fois).
5
Lance Claude Code et configure ta clé
Dans le terminal, tape claude puis dans Claude Code tape /config pour entrer ta clé API.
✓ Tu n'as pas de clé API ? → console.anthropic.com (compte gratuit)
1
Ouvre le Terminal
Appuie sur Cmd + Espace, tape Terminal, appuie sur Entrée.
💡 Le Terminal est l'app qui permet de taper des commandes texte sur ton Mac.
2
Télécharge le script d'installation
Clique sur le bouton ci-dessous pour télécharger le script Mac.
⬇️ Télécharger install-leo.sh
💡 Le fichier va dans ton dossier Téléchargements.
3
Lance le script dans le Terminal
Copie cette commande, colle-la dans le Terminal et appuie sur Entrée.
cd ~/Downloads && bash install-leo.sh
⏳ Le script installe Homebrew et Node.js automatiquement. Ça peut prendre 5 à 10 minutes la première fois. Laisse tourner.
4
Si une fenêtre "Xcode" s'ouvre
Clique "Installer" et attends. Le script attend automatiquement la fin de l'installation avant de continuer.
💡 Xcode Command Line Tools est un pack d'outils développeur requis sur Mac. Ça se passe une fois.
5
Ferme, rouvre le Terminal, tape claude
Quand le script affiche "Installation terminée", ferme le Terminal (Cmd+Q), rouvre-le, puis tape :
claude
✓ Configure ta clé API avec /config dans Claude Code.
Obtiens ta clé gratuite → console.anthropic.com

Prêt ? On commence.

Reviens sur la page d'accueil pour choisir ton profil et copier la commande d'installation.
