Claude répond
Authentification de NonChalanceApp Quand tu te connectes, voici ce qui se passe : 1. Tu entres ton email + mot de passe → Envoyé au serveur de manière sécurisée (HTTPS) 2. Le serveur vérifie ton identité → Ton mot de passe est comparé à une version chiffrée stockée en base de données 3. Un "badge" temporaire est créé → C'est un JWT (JSON Web Token), valable 7 jours → Il prouve que tu es bien toi sans que tu retapes ton mot de passe à chaque page ✓ Ton badge est stocké dans un cookie sécurisé. ✓ Il est invalidé si tu cliques "Déconnexion". Source : server/middleware/auth.js
